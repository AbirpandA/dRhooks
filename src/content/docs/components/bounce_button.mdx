## Overview

The BounceButton component is a playful and interactive button that adds subtle animation effects to enhance user interaction. It features a gentle bounce effect on hover and a satisfying scale reduction on click, making it perfect for calls-to-action and interactive elements that need a touch of personality.

### Key Features

- **Hover Animation**: Slight upward translation effect on hover for a floating appearance
- **Click Animation**: Smooth scale reduction on click for tactile feedback
- **Customizable**: Fully customizable through className props
- **Accessibility**: Built with native button semantics for optimal accessibility

## Key Concepts

### Animation Details

The button implements two main animation effects:
1. A slight upward translation (-0.5 units) on hover
2. A scale reduction to 90% on click/active state

These animations are achieved using CSS transforms and transitions, ensuring smooth and performant animations.

### Styling Architecture

The button uses a combination of:
- Base styles for consistent appearance
- Transition properties for smooth animations
- Transform properties for movement effects
- Hover and active states for interactivity
