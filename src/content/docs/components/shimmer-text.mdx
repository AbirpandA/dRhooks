## Usage

The `ShimmerText` component adds an animated shimmer effect to text content, creating an engaging visual experience. This effect is commonly used for loading states, highlighting important text, or adding visual interest to headings and content.


## Examples

<Steps>
  <Step>
    <strong>Basic Usage</strong>

    ```tsx
    <ShimmerText>Hello World</ShimmerText>
    ```
  </Step>

  <Step>
    <strong>Custom Duration and Width</strong>

    ```tsx
    <ShimmerText duration={3} shimmerWidth={20}>
      Loading...
    </ShimmerText>
    ```
  </Step>

  <Step>
    <strong>Custom Color (CSS or Tailwind)</strong>

    <p>You can set <code>--shimmer-color</code> on the component itself or any parent element. Two common approaches:</p>

    <em>Tailwind (arbitrary property)</em>

    ```tsx
    <ShimmerText className="[--shimmer-color:theme(colors.amber.400)] text-2xl font-bold">
      Golden Shimmer
    </ShimmerText>
    ```

    <em>Plain CSS class</em>

    ```css
    /* styles.css */
    .gold-shimmer { --shimmer-color: #FFD700; }
    ```

    ```tsx
    <ShimmerText className="gold-shimmer text-2xl font-bold">
      Golden Shimmer
    </ShimmerText>
    ```

    <em>Or set it inline on a parent:</em>

    ```tsx
    <div style={{ ['--shimmer-color' as any]: '#0ea5e9' }}>
      <ShimmerText>Blue shimmer via parent</ShimmerText>
    </div>
    ```

    The component provides sensible light/dark defaults, but you can override them anywhere using the `--shimmer-color` custom property.
  </Step>

</Steps>

## Props

<ComponentSource name="shimmer-text" />

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | Required | The text content to display with the shimmer effect |
| `className` | `string` | `undefined` | Additional CSS classes to apply to the text |
| `duration` | `number` | `2` | Duration of the shimmer animation in seconds |
| `shimmerWidth` | `number` | `10` | Width of the shimmer effect in percentage |
| `reverse` | `boolean` | `false` | Whether to reverse the animation direction |

## Implementation

<Callout>
  The ShimmerText component is built using Framer Motion for smooth animations and CSS gradients for the shimmer effect. The animation is achieved by smoothly transitioning the background position of a gradient overlay.
</Callout>

### Technical Details

- Built on `motion.span` from Framer Motion
- Uses CSS gradients for the shimmer effect
- Supports all standard HTML text attributes
- Maintains accessibility with decorative animations
